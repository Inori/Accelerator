Accelerator

Accelerator  一方通行，顾名思义，此项目目的为:

通过编写内存读写、HOOK代码，字体绘制代码等,实现一个动态汉化的框架，

使手动编写少量代码即可实现复杂的动态汉化。


目前包括以下两项，具体说明见各项目Readme.txt文档：

1. Accelerator

Accelerator主项目，实现字符串级的动态汉化。

收集游戏所有显示的字符串，通过FreeType DirectX GDI等功能库，自定义绘制函数替代游戏原有绘制函数。

实现 输出自定义字符串、字体选择、渲染等高度自由化定制功能。


2. Misaka

Accelerator子项目，实现文件级的动态汉化。

不改变游戏原有绘制函数，通过将脚本文件、图片文件等单文件从自定义封包中读取并复制到游戏内存，

实现汉化功能。
